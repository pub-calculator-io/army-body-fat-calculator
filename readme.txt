The Army Body Fat Calculator provides users with an accurate estimate of their body fat percentage using the standards of the U.S. Army.

[https://www.calculator.io/army-body-fat-calculator/](https://www.calculator.io/army-body-fat-calculator/)

== Usage ==

Add the Army Body Fat Calculator shortcode to your page, post or sidebar:

`[ci_army_body_fat_calculator]`

Add the following code to your template where you would like the Army Body Fat Calculator to appear:

`<?php display_ci_army_body_fat_calculator(); ?>`

1. Upload the Army Body Fat Calculator /ci_army_body_fat_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Army Body Fat Calculator plugin through the "Plugins" menu in WordPress.

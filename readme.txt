The Army Body Fat Calculator provides users with an accurate estimate of their body fat percentage using the standards of the U.S. Army.

[https://www.calculator.io/army-body-fat-calculator/](https://www.calculator.io/army-body-fat-calculator/)

__Army Body Fat Calculator Shortcode__

`[ci_army_body_fat_calculator]`

== Installation ==

1. Upload the /ci_army_body_fat_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the [Army Body Fat Calculator](https://www.calculator.io/army-body-fat-calculator/ "Army Body Fat Calculator Homepage") plugin through the "Plugins" menu in WordPress.

== Army Body Fat Calculator Usage ==

* Add the shortcode `[ci_army_body_fat_calculator]` to your page, post or sidebar.
* Add the following code: `<?php display_ci_army_body_fat_calculator(); ?>` to your template where you would like the Army Body Fat Calculator to appear.

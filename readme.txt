== Description ==

The Army Body Fat Calculator provides users with an accurate estimate of their body fat percentage using the standards of the U.S. Army.

== Installation ==

1. Upload the /ci_army_body_fat_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the [Army Body Fat Calculator](https://www.calculator.io/army-body-fat-calculator/ "Army Body Fat Calculator Homepage") plugin through the "Plugins" menu in WordPress.
3. Configure your settings via Appearance > Widgets > Army Body Fat Calculator.
4. And you're good to go!
